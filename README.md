This is a simple **.NET Console Application** built to practice and understand  
**Entity Framework Core**, **Code-First Migrations**, and basic **CRUD operations**.

- Query data using both **Fluent/Lambda syntax** and **LINQ Query syntax**
- Update records
- Delete records
- Display results in the console

- **.NET 8 Console Application**
- **Entity Framework Core**
- **SQL Server / LocalDB**
- **C#**

---

### Learning Purpose
This project is ideal for beginners who are learning:
- EF Core basics
- Code-First workflow
- CRUD operations
- LINQ querying
- Working with DbContext in .NET
